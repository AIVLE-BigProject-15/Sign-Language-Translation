사용한 딥러닝 기반 모델

CNN

컨볼루션 레이어가 많아질수록 영상의 특징을 더욱 견고하게 검출할 수 있습니다.

Local optimum에 빠지는 문제를 막기 위해 BatchNormalization을 사용했으며, 과적합을 방지하기 위해 Pooling과 Dropout을 사용하였습니다.

DNN

입력 데이터에 대해 연속적인 학습을 진행하기 위한 2개의 은닉층을 쌓았으며, 2D 이미지를 1D로 바꿔 학습했습니다.

LSTM

이전 단계에서 계산한 가중치를 현재 단계에 적용하여 계산하는 모델입니다.
Sequential Data에 특화되어 있지만 이미지 분류에 사용할 수도 있어 모델 비교를 위해 사용하였습니다.

딥러닝 모델 성능 비교

제공된 'sign_language.csv' 데이터로 학습했을 때의 모델 별 성능을 관찰한다.

CNN 모델을 적용하여 학습시킨 결과,

Deep Neural Network

Long Short-Term Memory
