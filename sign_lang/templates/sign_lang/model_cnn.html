수어 번역 서비스 모델

딥러닝 기반 수어(수화) 번역 모델 비교

고품질의 수어 번역 서비스를 개발함으로써 청각장애인의 정보 접근성을 개선하고, 궁극적으로는 청각장애인의 삶의 질과 사회적 참여를 높이는데 기여할 수 있습니다.

모델 설명

Deep Learning
LSTM

이전 단계에서 계산한 가중치를 현재 단계에 적용하여 계산하는 모델입니다.
Sequential Data에 특화되어 있지만 이미지 분류에 사용할 수도 있어 모델 비교를 위해 사용하였습니다.

Deep Learning
DNN

성능을 높이기 위해 입력데이터에 대해 연속적인 학습을 진행하기 위한 hidden layers를 쌓았으며 2D이미지를 1D로 바꾸기 위해 Flatten()을 사용하였습니다.

Deep Learning
CNN

컨볼루션레이어는 단계가 많아질수록 영상의 특징을 더더욱 견고하게 검출할 수 있으므로 레이어를 쌓았으며 이미지 학습이기 때문에 MaxPool2D을 사용해 특징을 뽑아내었고 BatchNormalization은 Local optimum 문제에 빠지는 가능성을 줄이기 위해서 사용했으며 과적합을 방지하기 위해 Dropout을 사용하였습니다.

모델 비교

노이즈를 제거해서 CNN 모델을 적용하였고
학습시킨 결과, 정확도는 0.99가 나왔습니다.
